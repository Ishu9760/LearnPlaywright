# Keyboard Shortcuts for JavaScript Development - Windows

## General Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl + C` | Copy |
| `Ctrl + X` | Cut |
| `Ctrl + V` | Paste |
| `Ctrl + Z` | Undo |
| `Ctrl + Shift + Z` | Redo |
| `Ctrl + A` | Select all |
| `Ctrl + F` | Find |
| `Ctrl + H` | Find & Replace |
| `Ctrl + /` | Toggle comment |
| `Shift + Alt + Down` | Duplicate line down |
| `Shift + Alt + Up` | Duplicate line up |
| `Alt + Down` | Move line down |
| `Alt + Up` | Move line up |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl + G` | Go to line |
| `Ctrl + P` | Quick file open |
| `Ctrl + Shift + P` | Command palette |
| `Ctrl + K, Ctrl + O` | Open folder |
| `Ctrl + Tab` | Switch between tabs |
| `Ctrl + Shift + Tab` | Switch tabs (reverse) |
| `Ctrl + \` | Split editor |
| `Ctrl + B` | Toggle sidebar |

## Code Editing

| Shortcut | Action |
|----------|--------|
| `Tab` | Indent |
| `Shift + Tab` | Outdent |
| `Ctrl + Shift + K` | Delete line |
| `Ctrl + Shift + Enter` | Insert line above |
| `Ctrl + Enter` | Insert line below |
| `Ctrl + L` | Select line |
| `Ctrl + D` | Select word (multi-select) |
| `Ctrl + Shift + L` | Select all occurrences |
| `Ctrl + Shift + \` | Jump to matching bracket |

## Debugging

| Shortcut | Action |
|----------|--------|
| `F9` | Toggle breakpoint |
| `F5` | Start debugging |
| `F10` | Step over |
| `F11` | Step into |
| `Shift + F11` | Step out |
| `Ctrl + Shift + D` | Open Debug view |
