## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

```bash
npm install @playwright/test
# or
yarn add @playwright/test
```

## 📖 Useful Shortcuts

### Quick Navigation
- `Ctrl + P` (Win) / `Cmd + P` (Mac) - Quick file open
- `Ctrl + Shift + P` (Win) / `Cmd + Shift + P` (Mac) - Command palette
- `Ctrl + G` (Win) / `Ctrl + G` (Mac) - Go to line

### Editing
- `Ctrl + /` (Win) / `Cmd + /` (Mac) - Toggle comment
- `Shift + Alt + Down` (Win) / `Option + Shift + Down` (Mac) - Duplicate line
- `Ctrl + Shift + K` (Win) / `Cmd + Shift + K` (Mac) - Delete line

## 🧪 Running Tests

```bash
# Run all tests
npx playwright test

# Run specific test file
npx playwright test Basics/001.js

# Run in debug mode
npx playwright test --debug
```
