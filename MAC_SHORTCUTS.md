# Keyboard Shortcuts for JavaScript Development - macOS

## General Editing

| Shortcut | Action |
|----------|--------|
| `Cmd + C` | Copy |
| `Cmd + X` | Cut |
| `Cmd + V` | Paste |
| `Cmd + Z` | Undo |
| `Cmd + Shift + Z` | Redo |
| `Cmd + A` | Select all |
| `Cmd + F` | Find |
| `Cmd + Option + F` | Find & Replace |
| `Cmd + /` | Toggle comment |
| `Option + Shift + Down` | Duplicate line down |
| `Option + Shift + Up` | Duplicate line up |
| `Option + Down` | Move line down |
| `Option + Up` | Move line up |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl + G` | Go to line |
| `Cmd + P` | Quick file open |
| `Cmd + Shift + P` | Command palette |
| `Cmd + K, Cmd + O` | Open folder |
| `Control + Tab` | Switch between tabs |
| `Control + Shift + Tab` | Switch tabs (reverse) |
| `Cmd + \` | Split editor |

## Code Editing

| Shortcut | Action |
|----------|--------|
| `Tab` | Indent |
| `Shift + Tab` | Outdent |
| `Cmd + Shift + K` | Delete line |
| `Cmd + Shift + Enter` | Insert line above |
| `Cmd + Enter` | Insert line below |
| `Cmd + L` | Select line |
| `Cmd + D` | Select word (multi-select) |
| `Cmd + Shift + L` | Select all occurrences |
| `Cmd + Shift + \` | Jump to matching bracket |

## Debugging

| Shortcut | Action |
|----------|--------|
| `F9` | Toggle breakpoint |
| `F5` | Start debugging |
| `F10` | Step over |
| `F11` | Step into |
| `Shift + F11` | Step out |
| `Cmd + Shift + D` | Open Debug view |
